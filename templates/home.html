Introduction

About:
This is a mixed aptitude test. This test consists of 30 questions with a time limit of 15 minutes. These assessments are designed to evaluate a candidate's cognitive abilities, problem-solving skills, and communication skills.

Instructions:
Each question contains 4 answers, and the user has to select on the one among 4 options. The user can only select one answer per question. The user can only move to the next question after selecting an answer. The user can not go back to the previous question. The user can not skip a question.

When you are ready, click "Begin" to start the test.
